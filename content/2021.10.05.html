fossil.exe: 下載

stunnel.exe: 下載

將.exe設在可攜系統的path

start /MIN %Disk%:\tmp\stunnel\bin\stunnel.exe

在命令列輸入
python -m  http.server 8000 --bind 127.0.0.1
可啟動www server 讓使用者透過http://localhost:8000/擷取到此目錄的檔案
操作二
將操作一 python -m http.server 換成Fossil SCM網路服服務器
建立fossil倉儲pj2022.fossil
fossil init pj2022.fossil
修改密碼
fossil user password username new_password -R pj2022.fossil
username 輸入原本的使用者
new_password 輸入新密碼
-R pj2022.fossil 為倉儲檔案
fossil server -P 127.0.0.1:9000 --https y:\tmp\pj2022.fossil
以上指令啟動後 因以 --http 送出伺服資料 因此http://localhost:9000/並無法連線 需在stunnel https代理伺下才能連線
設定stunnel讓443可以接受Fossil SCM從 port9000送出資料
將\stunnel\config\stunnel.conf
[https]
; 配合從此命令轉接資料: fossil.exe server -P 127.0.0.1:9000 --https y:\tmp\pj2022.fossil
accept  = 443
connect = 9000
cert = localhost.crt
key = localhost.key
TIMEOUTclose = 0
重啟後https://localhost/就可以看到Fossil SCM網際介面
